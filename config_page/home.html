How this work

The EcoClass system is an interconnected network of ESP8266 microcontrollers designed to monitor and control environmental conditions across multiple units. Each sub-unit is equipped with sensors to measure temperature, humidity, gas levels, and detect motion or light status. These sub-units use static IP configurations to communicate with a central main unit over a Wi-Fi network. The main unit aggregates sensor data from the sub-units and can also send commands or updated configurations, such as Wi-Fi credentials, to the sub-units. Additionally, a dedicated display unit for showing time, also using a static IP, receives real-time data from the main unit and visually presents the current environmental readings and statuses on an OLED screen. This system ensures efficient monitoring, centralized control, and a clear visual interface for managing smart climate operations.
